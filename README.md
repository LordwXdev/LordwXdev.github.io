# Lord Portfolio 2026

A high-performance, modern developer portfolio built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. Featuring a dynamic bubble hero section, interactive project filtering, and a clean, responsive design.

## 🚀 Features

* **Interactive Hero Section:** Custom HTML5 Canvas bubble animation with a typewriter effect.
* **Service Grid:** Showcases expertise with custom Lucide icons.
* **Project Portfolio:** Category-based filtering (Web, App, Design) for work samples.
* **Professional Timeline:** A vertical experience tracker for career history.
* **Testimonials Slider:** Interactive client feedback carousel.
* **Optimized Performance:** Built with Next.js App Router for superior speed and SEO.
* **Dynamic Blog:** Ready-to-use blog grid and dynamic project detail pages.

## 🛠️ Tech Stack

* **Framework:** Next.js 15 (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
* **Deployment:** Optimized for Vercel
